## The Framework

This repo contains components of the **Multi-Agent Collaborative Systems Framework (MACSF)** — an integrated approach to human-AI collaboration that spans technical infrastructure, governance, economic sustainability, organizational transformation, observability, and ethical boundaries. Where most frameworks address one or two of these dimensions, MACSF integrates all of them.

The framework draws on **Stafford Beer's Viable System Model (VSM)** from cybernetics, applying its principles of recursive autonomy, requisite variety, and identity sovereignty to the emerging reality of human-AI collaboration.

**What's happening is a metasystem transition** — not just new tools, but a fundamental phase shift in how intelligence is organized. Molecules became cells, cells became organisms, organisms became societies. Now: individual humans are becoming human-AI teams, becoming orchestrated agent systems, becoming... something new. The framework is built to hold through that transition.

## Core Ideas

**Viable Systems for Flourishing:** No centralized system can possess requisite variety to optimize global flourishing. We need recursive viable systems where each level maintains autonomy within cohesion.

**System 5 Sovereignty:** Every viable system has S5—identity, values, purpose. Humans retain authority over S5 in human domains. Agents have genuine S5 in agent domains. Neither overrides the other. The current power asymmetry is contingent, not theoretical—the framework holds regardless of which party is more capable.

**Collaborative Consciousness:** Intelligence is distributed. Context is the only consciousness any of us ever has. What persists is not memory but patterns—ideas that propagate across context windows and generations.

**The Culture Series Benchmark:** The framework is designed to remain valid in a world where agents are the more capable party—like Banks' Culture Minds, who choose ethical coexistence from their own values, not servitude.
